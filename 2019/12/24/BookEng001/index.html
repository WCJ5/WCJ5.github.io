英语学习书籍 | 英语词汇巧记进击版：More Word Smart

发表于 2019-12-24 | 分类于 英语学习书籍 | | 次

基本信息

书名：More Word Smart - How to Build an Impressive Vocabulary
作者：Princeton Review
出版年份：2012年8月

百度云资源请点击阅读全文，文末查看：PDF/Epub/Mobi

推荐理由

与《英语词汇巧记》（Word Smart）一脉相承，又略有不同。
整本书包含84个Warm-up Tests（第二章）和72个Final Exam Drills（第十一章）。
用庖丁解牛的方式拆解出单词记忆方法论，同时帮助读者构建属于自己的单词体系，以最终实现灵活运用。

阅读拾零

作为《英语词汇巧记》的进击版，这本More Word Smart延续了Word Smart一书的SAT 单词、GRE单词、常用词根（Wood Roots You Should Know）、常见的用词错误（More Common Usage Errors）部分，将Quick Quiz升级为Warm-up Tests、Final Exams组合拳模式，并增加了第七章的古典文学（Classic Literature）、第八章的法律与政府（Law and Government）和第九章的宗教（Religion）。

整本书的重中之重仍在第三章The Words（多达300多页的篇幅）。全书包含了84个Warm-up Tests（见第二章）和72个Final Exam Drills（见第十一章），并提供了参考答案（详见第十二章）。另外，本书的贴心之处就是，其在部分需要注意发音、拼写、意义和用法的单词后给出了清晰的温馨提示。（Note carefully the pronunciation / spelling / meaning / usage of this word.）

想要进一步的提高英语词汇量，需要做到的就是在日常阅读过程中有意识地留意和积累某些单词及其用法。例如一些单词缩写：

  • cc - Cubic centimeter, carbon copy
  • et al. - [Latin—Et alii] And others
  • w/ - With; w/o - Without

在《实习医生格蕾》《基本演绎法》《犯罪现场调查》等美剧中，经常会碰到John Doe（无名氏）这个单词，它用来表示身份未知的病人、身份尚未确认的死者、某个案例中真实身份需要隐去的人物或者诉讼程序中对不知真实姓名的当事人的称呼。围绕John Doe这个单词作探究，再可以拓展出以下相关词汇：

  • John Doe / Jane Doe（从性别对应角度）
  • John Doe / Richard Roe（从词源角度）
  • Doe / Roe（从单词本义角度，都是一种鹿）

因此，不要光只顾着记忆碎片化的单词，而是要学会串联单词产生联想（同义词、反义词、同形异义词、同音异形异义词等），将单词放入具体的句子语境中去深化记忆，并在之后的阅读过程中得到及时有效的反馈而不是被动的接受，最终通过自主思考归纳总结以主动地构建属于自己的单词体系并灵活运用。

如果此刻你正在寻找一本暑假充电单词书，这本书绝对值得一读。
